This project mimics Apple music application. You are able to search for songs in the database, add songs to the database and more. This project utilized the knowledge of Java Collections, File I/O, Exceptions, Objects and classes, inheritance, and interfaces. The following functions below are some functions showcasing what the application can do.

SEARCH: Searches store for an audio content with specified title

SEARCHA: Searches store for audio content with the specified artist name

SEARCHG: Searches store for audio content with the specified genre

DOWNLOAD: Downloads specific audio content from store

DOWNLOADA: Downloads all audio content associated with artist name

DOWNLOADG: Downloads all audio content associated with genre name
